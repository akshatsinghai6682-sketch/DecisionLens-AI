# DecisionLens AI - Frontend

React + TypeScript + Vite frontend for DecisionLens AI decision simulator.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
npm run build
```

## Project Structure

- `src/components/` - Reusable React components
- `src/pages/` - Page components
- `src/hooks/` - Custom React hooks
- `src/context/` - React Context providers
- `src/utils/` - Utility functions

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- ShadCN UI
- Recharts
- React Router
